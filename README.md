 Wine Quality Prediction

Machine learning project for predicting red wine quality using regression models and model interpretability techniques.

## Dataset

This project uses the Wine Quality (Red Wine) dataset from the UCI Machine Learning Repository.

## Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regressor (SVR)
- K-Nearest Neighbors (KNN)

## Libraries

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SHAP
